# Sales-Profit-Analysis-PowerBI

## Objective
To analyze sales, profit, and logistics data using Excel and Power BI in order to
identify revenue trends, profitability drivers, and operational inefficiencies.

## Tools Used
- Power BI
- Microsoft Excel

## Key Analysis Performed
- Sales and revenue trend analysis
- Profit and margin analysis
- Cost impact on profitability
- Logistics and shipping performance review

## Dashboards Created
- Revenue Overview Dashboard
- Profit & Cost Analysis Dashboard
- Regional and Product Performance Dashboard

## Key Insights
- Identified high-revenue but low-margin products
- Highlighted regions with high logistics costs
- Provided insights to improve profit margins

## Outcome
Delivered actionable insights to support business decision-making and improve
overall operational efficiency.
